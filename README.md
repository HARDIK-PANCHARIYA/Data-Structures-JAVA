# Data Structures and Algorithms in Java

This repository contains my practice implementations of various data structures, along with sorting and searching algorithms, in Java. The purpose is to gain a deep understanding of these foundational concepts while improving proficiency in Java.

## Topics Covered

### Data Structures

- **Lists**
  - ArrayList
  - LinkedList
  - Vector

- **Sets**
  - HashSet
  - LinkedHashSet
  - TreeSet

- **Maps**
  - HashMap
  - LinkedHashMap
  - TreeMap

### Algorithms

- **Sorting Algorithms**
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort
  - Heap Sort

- **Searching Algorithms**
  - Linear Search
  - Binary Search
